## Determinazione del Security Level Target

### 1. Introduzione

Questo capitolo illustra la metodologia per determinare il Security Level Target (SL-T) appropriato per ogni zona di sicurezza, elemento critico nella gestione del rischio secondo IEC 62443.

### 2. Contenuti Principali

- Scala dei Security Levels (SL0-SL4)
- Criteri di determinazione del SL-T
- Valutazione delle conseguenze (Impact Assessment)
- Valutazione della probabilità di esposizione
- Matrici di rischio
- Documentazione e gestione delle decisioni di SL
